# Sample Conversations

These sample conversation scripts help you test the chatbot's
memory and multi-turn conversation capabilities.

## How to use

1. Start the app: streamlit run ph1/app/main.py
2. Open the Chat page
3. Ask questions one by one from any sample file
4. Observe how the chatbot remembers context across messages

## What to look for

- Follow-up questions answered without repeating context
- References to earlier parts of the conversation
- Coherent multi-turn dialogue

## Files

- aws-basics.txt — Tests memory across EC2 and pricing topics
- serverless-basics.txt — Tests memory across Lambda concepts
